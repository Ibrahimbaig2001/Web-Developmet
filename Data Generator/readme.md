To achieve this follow the steps

Step 1: Initialize npm in the project using the command
```` npm init -y ````

Step 2: Load express4 in the project using the command
````npm i express@4````

Step 3: Install ejs in the project using the command

````npm i ejs````

Step 4: ```npm i mongoose```

Step 5: To use ejs with express create a folder name ''views'' and add a file name index.ejs in it. Extract the content from bootstrap.

Step 6:To start running the server use the following command ``` nodemon main.js```
